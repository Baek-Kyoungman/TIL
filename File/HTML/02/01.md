# 01. 블록(Block), 인라인(Inline)

1. **블록** 요소
  - DIV, H1, P
  - 사용 가능한 최대 가로 너비를 사용한다.
  - 크기를 지정할 수 있다. 
  - (width:100%; height:0;)로 시작
  - 수직으로 쌓임
  - margin, padding 위, 아래, 좌, 우, 사용 가능
1. **인라인** 요소
  - SPAN, IMG
  - 필요한 만큼의 너비를 사용한다. 
  - 크기를 지정할 수 없다.
  - (width:0; height:0;)로 시작
  - 수평으로 쌓임
  - margin, padding 위, 아래는 사용할 수 없다.
